# **App Name**: WebGeniusAI

## Core Features:

- AI-Powered Website Generation: Generates multi-page, responsive websites from natural language prompts using GPT models like GPT-4o.
- Smart Prompt Assistant: The AI-powered tool asks clarifying questions and optimizes prompts to improve website generation quality using GPT models like GPT-4o.
- Logo-Aware Theming: Extracts dominant colors from uploaded logo images and applies them to the website's color scheme using GPT models like GPT-4o.
- Real-Time Preview Panel: Displays the generated website instantly without manual coding.
- Live Code Editor: Allows users to tweak the generated code and immediately preview changes.
- Section-Level Regeneration: Enables users to modify specific sections of the website, such as hero text or color theme using GPT models like GPT-4o.
- Downloadable Project: Provides the option to download the complete project as a ZIP file containing all generated files and folders.

## Style Guidelines:

- Primary color: A saturated, vivid purple (#9C27B0) to reflect the advanced AI capabilities and creativity.
- Background color: A very light grayish-purple (#F3E5F5) for a clean, modern aesthetic.
- Accent color: A contrasting blue-violet (#3F51B5) to highlight key UI elements and CTAs.
- Body font: 'Inter', a sans-serif typeface known for its clean and modern appearance, ideal for ensuring readability and a high-tech aesthetic.
- Headline font: 'Space Grotesk', for a computerized, techy feel; great for drawing attention to headlines and short amounts of text.
- Use clean, minimalist icons to represent different website sections and functionalities.
- Subtle, elegant transitions and animations to enhance user experience.
